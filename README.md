# 96 Clothing Brand Website

This project is a modern, high-fashion landing page for the clothing brand "96".

## How to View a Locally

1. Open the `index.html` file in your preferred web browser (e.g., Chrome, Firefox, Safari).
2. The website is responsive and should look good on all devices.

## Features

- **Responsive Design**: Adapts well to mobile, tablet, and desktop screens.
- **Scroll Animations**: Smooth fade-in effects as you scroll down.
- **Modern Typography**: Uses 'Space Grotesk' for headings and 'Inter' for body text.
- **Interactive Elements**: Hover effects on buttons, products, and navigation links.

## Note on Images

Due to a temporary service limitation, the images in the website are currently using stylized CSS placeholders (gradients and solid colors). You can replace these placeholders with real product photos in the future by updating the HTML and CSS.

## File Structure

- `index.html`: The main page content.
- `styles.css`: All the styling rules.
- `script.js`: Handles the interactive animations.

Enjoy exploring the brand "96"!
